webcomponents
=============
web components学习中，这里放一些demo

demo查看方式

```
$ git clone git@github.com:LingyuCoder/webcomponents.git
$ cd webcomponents
$ npm install -g koa-anywhere
$ ka
```
然后选择子目录下的index.html即可

**请用最新版的chrome打开
